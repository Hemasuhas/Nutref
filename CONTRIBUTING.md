# Contributing to Nutref

Thank you for considering contributing to the Nutref project! We appreciate your help in improving our project.

## Code of Conduct
We ask that you adhere to our code of conduct to ensure a welcoming and supportive environment. Please read our [Code of Conduct](CODE_OF_CONDUCT.md) for more details.

## Reporting Bugs
If you encounter a bug, please follow these steps to report it:
1. Check if the bug has already been reported.
2. Provide a clear and descriptive title for your issue.
3. Describe the steps to reproduce the bug, including any relevant information like operating system, libraries used, etc.

## Suggesting Features
We welcome suggestions for new features! To suggest a feature:
1. Open a new issue with a descriptive title.
2. Explain the purpose of the feature and how it may improve the project.

## Submitting Code Changes
If you want to contribute code:
1. Fork the repository on GitHub.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them.
4. Push your changes to your fork.
5. Open a pull request to the main repository.

Make sure to reference any open issues that your pull request addresses.

## Development Setup
To set up the development environment:
1. Clone the repository: `git clone https://github.com/Hemasuhas/Nutref.git`
2. Navigate to the project directory: `cd Nutref`
3. Install the required dependencies: `npm install` or your preferred package manager.

## Testing Requirements
To ensure code quality and functionality:
1. Run the tests using the command: `npm test`
2. Ensure all tests pass before submitting your pull request.

## Support Information
For any questions or support, feel free to reach out:
- Open an issue on GitHub.
- Contact us via [email](mailto:support@nutref.com).

Thank you for your contributions!
